Key design elements in this mockup:

  • Top color bar on cards — green gradient for positive, red for negative (replaces colored borders)
  • Summary stats row at top — quick portfolio overview before cards
  • Pill-shaped metric badges — cashflow on tinted backgrounds instead of plain text
  • Soft shadows instead of borders — cards float above the gray background
  • Rounded corners (16px) — softer, more modern feel
  • Indigo accent for primary actions + CoC, green for cashflow
  • Muted delete — not red, just faded text (less visual noise)
